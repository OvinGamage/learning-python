## 🌟 Featured Capstone Project
### Engineering Material Stress Analysis Pipeline
**File:** `/engineering_data_pipeline.py`

This project marks my transition from a "learner" to a "builder." It is an end-to-end data pipeline that automates the ingestion, mathematical cleaning, and visualization of structural stress-test data across multiple active manufacturing projects.

#### 🛠️ Key Technical Features Implemented:
* **Defensive Data Ingestion (Pandas):** Programmed an automated check to flag corrupted negative sensor readings (`-999.0`). Successfully implemented an imputation matrix to replace anomalies with the true dataset mean without dropping rows or data bias.
* **Vectorized Multi-Metric Aggregation:** Utilized high-performance `.groupby().agg()` indexing to process testing volumes and structural failure thresholds concurrently by material class.
* **Automated Engineering Chart Hygiene (Matplotlib):** Designed an automated visual report output engine featuring professional baseline constraints (custom grids, exact scale boundaries, high-DPI tracking, and programmatic axis allocation).
